# Dot.jsx
_React Component_

A nice 16x16 coloured dot.

### Props

| Prop name     | Type       						  | Description 										|
| ------------- |:-----------------------------------:| --------------------------------------------------- |
| [className]   | _string_      					  | class to use instead of the default					|
| [color]       | _oneOf 'red', 'green', 'yellow'_    | dot color		 									|
